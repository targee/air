# 如何申請帳號？

### Hello，歡迎來到[「PDIS公開討論區」](https://talk.pdis.nat.gov.tw/)，首頁畫面如下：

<img src="/uploads/default/original/1X/c78f778b557354529d4dd03cf1b733ec58e94fea.PNG" width="690" height="484">

1. 點選右上角「註冊」

    <img src="/uploads/default/original/1X/c30fee4589225f4f97e925355bc04ee01ff4e1b9.png" width="277" height="142">

2. 輸入電子郵件、用戶名稱、名稱及密碼，確認無誤後按下「建立新帳號」

    <img src="/uploads/default/original/1X/5269bfcdad470f48ecd9e64eb620aa66f58e6897.PNG" width="690" height="416">

3. 建立帳號後待系統發出確認信，收到確認信後按下連結至啟用帳戶頁面

    <img src="/uploads/default/original/1X/33e4e2736e15a4055eca8bfc86ab39ee081592b7.png" width="682" height="237">

4. 按下「點擊此處啟用你的帳戶」

    <img src="/uploads/default/original/1X/2d4f7789a84c4626b71dc9e1903f2b8c84954408.png" width="352" height="121">

5. 恭禧！你現在是[「PDIS公開討論區」](https://talk.pdis.nat.gov.tw/)的成員了！

# 如何進入「前瞻基礎建設QA」討論區？

### 新申請帳號若沒加入「前瞻基礎建設QA」群組無法進入「前瞻基礎建設QA」討論區，待管理員將成員加入群組後即可進入

<img src="/uploads/default/original/1X/908561cb815dcfb3da132946e03568efc0bc9441.PNG" width="301" height="394">

# 如何將成員加入「前瞻基礎建設QA」群組？

1. 管理員於右上角點選「管理員」

    <img src="/uploads/default/original/1X/2ce9cc76e5689c5388d88dc1c3ff1c35a1eceb68.png" width="343" height="500">

2. 於上方點選「用戶」，再選擇成員名稱

    <img src="/uploads/default/original/1X/ef8508e1a174ca3405c5c3485f93a774fef9cb3c.PNG" width="690" height="450">

3. 將「板主？」選擇「是」，「信任等級」選擇「2 - 成員」，「客製」加入「QAGroup」

    「板主」：給予成員刪除文章的權限
    「信任等級」：發表文章不需經過管理員審核
    「客製」：給予成員進入「前瞻基礎建設QA」討論區的權限

    <img src="/uploads/default/original/1X/4310c0d1b0b829decd6eeae6fa65ef860df02800.PNG" width="690" height="462">

4. 現在成員可進入「前瞻基礎建設QA」討論區並發表文章了！

# 如何在「前瞻基礎建設QA」討論區發表文章？

1. 於首頁「所有分類」裡選擇「前瞻基礎建設QA」

    <img src="/uploads/default/original/1X/a17d4c4626f9eac921c71db082c88ce7a94d849b.png" width="316" height="400">

2. 點選右上角「新討論話題」

    <img src="/uploads/default/original/1X/c68606cbfffade4ece02df6fe661cac7776bce3e.PNG" width="690" height="294">

3. 輸入「文章標題」、「文章內文」後按下「建立討論話題」即可發表文章

    <img src="/uploads/default/original/1X/6f37b5cf337dd081751bf04f44ad190f74f88546.png" width="690" height="168">

4. 發表文章成功！

    <img src="/uploads/default/original/1X/80f2203c6731175894dc02a2207a33fb4a075a00.png" width="690" height="346">

# 如何修改文章？

1. 點選「編輯」按鈕

    <img src="/uploads/default/original/1X/b4f153f094ff1e60b41ead22848b8879980badf9.png" width="690" height="346">

2. 修改「文章標題」、「文章內文」後按下「儲存編輯」

    <img src="/uploads/default/original/1X/54a9ef85a2ae3691eb4aa4604fa8ca0a4c67f10e.png" width="690" height="178">

3. 修改文章成功！

    <img src="/uploads/default/original/1X/91a187384129993bfe9d8af7df06ff120995ba49.png" width="690" height="350">

# 如何分享文章

    <img src="/uploads/default/original/1X/cfe0f51a04ceaa746dece88059c8ed3de539f516.png" width="690" height="346">
